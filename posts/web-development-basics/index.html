Web 開發基礎：HTML、CSS、JavaScript 入門

現代網頁開發的三大支柱：HTML 提供結構、CSS 負責樣式、JavaScript 添加互動性。本文將帶你了解這三種技術的基礎知識。

HTML：網頁的骨架

基本結構

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>歡迎來到我的網站</h1>
    <p>這是一個段落。</p>
</body>
</html>

常用標籤

  • <h1> 到 <h6>：標題標籤
  • <p>：段落
  • <a>：連結
  • <img>：圖片
  • <div>：容器
  • <span>：行內容器

CSS：讓網頁變美麗

基本語法

選擇器 {
    屬性: 值;
    屬性: 值;
}

選擇器類型

元素選擇器

p {
    color: blue;
    font-size: 16px;
}

類別選擇器

.highlight {
    background-color: yellow;
}

ID 選擇器

#header {
    background-color: #333;
    color: white;
}

CSS 盒模型

每個 HTML 元素都是一個盒子，包含：

  • Content：內容
  • Padding：內邊距
  • Border：邊框
  • Margin：外邊距

JavaScript：添加互動性

基本語法

// 變數宣告
let message = "Hello, World!";
const PI = 3.14159;

// 函數
function greet(name) {
    return "Hello, " + name + "!";
}

// 事件處理
document.getElementById("button").addEventListener("click", function() {
    alert("按鈕被點擊了！");
});

DOM 操作

// 選取元素
const element = document.getElementById("myElement");
const elements = document.querySelectorAll(".myClass");

// 修改內容
element.innerHTML = "新的內容";
element.textContent = "純文字內容";

// 修改樣式
element.style.color = "red";
element.classList.add("newClass");

實戰練習：建立簡單網頁

HTML 結構

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>計數器應用程式</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>計數器</h1>
        <div class="counter">
            <span id="count">0</span>
        </div>
        <button id="increase">+1</button>
        <button id="decrease">-1</button>
        <button id="reset">重設</button>
    </div>
    <script src="script.js"></script>
</body>
</html>

CSS 樣式

body {
    font-family: Arial, sans-serif;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    margin: 0;
    background-color: #f0f0f0;
}

.container {
    text-align: center;
    background: white;
    padding: 2rem;
    border-radius: 10px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

.counter {
    font-size: 3rem;
    font-weight: bold;
    margin: 1rem 0;
    color: #333;
}

button {
    font-size: 1.2rem;
    padding: 0.5rem 1rem;
    margin: 0.5rem;
    border: none;
    border-radius: 5px;
    background-color: #007bff;
    color: white;
    cursor: pointer;
    transition: background-color 0.3s;
}

button:hover {
    background-color: #0056b3;
}

JavaScript 邏輯

let count = 0;
const countElement = document.getElementById('count');
const increaseBtn = document.getElementById('increase');
const decreaseBtn = document.getElementById('decrease');
const resetBtn = document.getElementById('reset');

function updateDisplay() {
    countElement.textContent = count;
}

increaseBtn.addEventListener('click', () => {
    count++;
    updateDisplay();
});

decreaseBtn.addEventListener('click', () => {
    count--;
    updateDisplay();
});

resetBtn.addEventListener('click', () => {
    count = 0;
    updateDisplay();
});

進階主題

響應式設計

/* 手機版 */
@media (max-width: 768px) {
    .container {
        padding: 1rem;
    }
    
    button {
        display: block;
        width: 100%;
        margin: 0.5rem 0;
    }
}

// 箭頭函數
const add = (a, b) => a + b;

// 解構賦值
const {name, age} = person;

// Promise
fetch('/api/data')
    .then(response => response.json())
    .then(data => console.log(data));

總結

HTML、CSS 和 JavaScript 是現代網頁開發的基礎。通過持續學習和實踐，你可以建立出功能豐富、視覺吸引人的網站。記住，程式設計是一個持續學習的過程，保持好奇心和耐心是成功的關鍵！
